BOSTIn (Broad Overview of Sequence & Topology Incongruence) is a new, user-friendly phylogenetic artifact identification package.

It takes an aligned fasta file and uses up to three different metrics to assess the alignment for compositional heterogeneity, site saturation and branch length heterogeneity.
In addition, it then produces a unique "narrative report", which tells you which taxa are potentially concerning, and why. It also presents possible next steps to ameliorate those artifacts, or explore them in more detail.
For Compositional Heterogeneity, BOSTIn uses nRCFV, which measures how variable nucleotide or amino acid composition is across the dataset. While all three measurements give taxon specific ratings as well as whole dataset ratings, nRCFV also gives a character specific measurement.
For Site Saturation, BOSTIn uses the DEScore for amino acid data and the C-Score for nucleotide data. The former measures the proportion of between Dayhoff category changes across a dataset, while the latter compares the transition/transversion ratio to the distance between genes in the dataset (the uncorrected p-distance)
For Branch Length Heterogeneity, BOSTIn uses the LB-Score, which compares the variation of distances between taxa across the tree.
To find out more about how all these metrics are calculated, you can visit our citations page here

To get started, simply upload an aligned fasta file below, note whether it is nucleotide or amino acid data, and then tell BOSTIn which artifacts you are interested in exploring today by clicking the text boxes. BOSTIn can run all three analyses at once, or any combination of the three separately!
